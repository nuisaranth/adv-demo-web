# 🚀 Team Digital: เว็บไซต์ Demo สำหรับคอร์สเรียน

เทมเพลตเว็บไซต์สำเร็จรูปสำหรับคอร์ส AI & Digital Marketing โหลดไว รองรับ SEO เต็มรูปแบบ ออกแบบมาให้นักเรียนเปิดเว็บไซต์ธุรกิจของตัวเองได้ภายใน 2 นาที!

---

## 🛠 ขั้นตอนที่ 1: สร้างเว็บไซต์ของคุณ (Deploy)
คลิกที่ปุ่มด้านล่างเพื่อคัดลอกโปรเจกต์นี้ไปยังบัญชี GitHub ของคุณ และเปิดใช้งานเว็บไซต์ให้ออนไลน์ผ่าน Vercel ทันที

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/nuisaranth/adv-demo-web)

---

## 📝 ขั้นตอนที่ 2: ตั้งค่าข้อมูลร้านค้าและ SEO (ทำผ่านเว็บ GitHub ได้เลย)
คุณสามารถแก้ไขข้อมูลเว็บไซต์ได้ง่ายๆ โดยเข้าไปแก้ไขไฟล์ต่อไปนี้:

### 1. ไฟล์ `src/config.yaml`
* **ตั้งชื่อแบรนด์ (Identity):** แก้ไขชื่อแบรนด์และใส่ลิงก์เว็บไซต์จริงของคุณ
* **ตั้งค่า SEO (Metadata):** แก้ไขหัวข้อ (Title) และคำอธิบาย (Description) ที่ต้องการให้โชว์บนหน้า Google

### 2. ไฟล์ `src/navigation.ts`
* **จัดการเมนู (Navigation):** แก้ไขชื่อเมนูบนแถบนำทางด้านบน และข้อมูลการติดต่อด้านล่างเว็บไซต์ (Footer)

---

## ✍️ ขั้นตอนที่ 3: การเขียนบทความ (Blog)
หากต้องการเพิ่มหรือแก้ไขบทความ ให้เข้าไปที่โฟลเดอร์: 
👉 `src/content/post/`

* คุณสามารถกดคัดลอกไฟล์ตัวอย่าง (ที่มีนามสกุล `.md`) แล้วแก้ชื่อไฟล์และเนื้อหาด้านในเป็นบทความใหม่ของคุณเองได้เลย

---

## 💻 สำหรับผู้พัฒนา (นำไปรันบนเครื่องตัวเอง)
หากคุณต้องการแก้ไขเชิงลึกผ่าน VS Code ให้ใช้คำสั่งเหล่านี้ใน Terminal:

```bash
npm install    # ติดตั้งระบบและ Dependencies
npm run dev    # รันเว็บเซิร์ฟเวอร์เพื่อดูผลลัพธ์ (http://localhost:4321)
npm run build  # สร้างไฟล์ Static สำหรับอัปโหลดขึ้นเซิร์ฟเวอร์จริง
